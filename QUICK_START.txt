📦 INSTALLATION (5 MINUTES):

1. Install dependencies:
   npm install

2. Set up Twilio SMS (optional but recommended):
   - Sign up: https://www.twilio.com/try-twilio  
   - Get FREE credits for testing
   - Copy config.example.js to config.js
   - Add your Twilio credentials
   - Change admin password!

3. Start the server:
   npm start

4. Test it:
   Customer form: http://localhost:3000
   Admin dashboard: http://localhost:3000/admin
   Login: admin / admin123 (CHANGE THIS!)

═══════════════════════════════════════════════════════════
  📱 HOW IT WORKS - SIMPLE VIEW
═══════════════════════════════════════════════════════════

CUSTOMER SIDE:
1. Scans QR code → Opens form on phone
2. Fills form (30 seconds with dropdowns!)
3. Gets SMS: "We received your request"
4. Gets SMS: "Your quote is $45"
5. Replies YES or NO
6. Gets SMS: "Confirmed! See you at 2 PM"

YOUR SIDE:
1. Login to http://yourwebsite.com/admin
2. See new request
3. Click "Send Quote" → Enter $45
4. Customer gets SMS automatically
5. Click "Confirm" when they accept
6. Click "Complete" after ride done

═══════════════════════════════════════════════════════════
  📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════

SMS not sending?
→ Check Twilio credentials in config.js
→ Verify phone number has country code (+1...)
→ Check Twilio console for error messages

Can't login to dashboard?
→ Check config.js for correct username/password
→ Default is admin/admin123

Form not submitting?
→ Make sure server is running (npm start)
→ Check browser console (F12) for errors

Database errors?
→ Make sure app has write permissions
→ Delete ride_requests.db and restart (creates new)
